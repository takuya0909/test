# git innt
    ディレクトリ内に.gitディレクトリを作成する(ローカルリポジトリ))

# git status
    ローカルリポジトリの状態の確認

# git diff
    ワークツリーとステージエリアの差分を確認。

# git add [ファイル名]
  　ワークツリーにあるファイルをステージエリアに登録

# git add .
    リポジトリ内のファイル、フォルダをすべてステージエリアに登録

# git rm [削除したいファイル]
    ファイルを削除

#git rm -r [削除したいディレクトリ]
    削除したいディレクトリ

# git clone [github のSSHのURL]
　　GitHub上のリポジトリをローカルリポジトリ（自分の端末）に複製する
